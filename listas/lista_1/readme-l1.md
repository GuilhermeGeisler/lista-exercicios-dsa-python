# 🐍 Fundamentos de Python - Lista 1 de Exercícios (DSA)

## 🌟 Sobre o Projeto  
Lista de exercícios desenvolvida durante o curso **"Fundamentos de Linguagem Python - Do Básico a Aplicações de IA"** da **Data Science Academy**. O objetivo é praticar os conceitos fundamentais da linguagem Python, desde operações básicas até estruturas de dados como listas, tuplas, dicionários e conjuntos.

- 🎯 30 exercícios práticos abordando os principais fundamentos da linguagem
- 📓 Desenvolvido em Jupyter Notebook para melhor visualização e execução interativa
- 🧠 Conceitos trabalhados: variáveis, tipos de dados, operadores, estruturas condicionais, manipulação de strings, listas, tuplas, dicionários e conjuntos
- ✅ Todos os exercícios resolvidos com exemplos práticos

## 🔥 Destaques Técnicos

- **Exploração de Tipos de Dados:** Abordagem prática de inteiros, floats, strings, booleanos, listas, tuplas, dicionários e conjuntos.
- **Manipulação de Strings:** Uso dos principais métodos como `strip()`, `upper()`, `replace()` e fatiamento.
- **Estruturas de Dados:** Aplicação de operações específicas de cada estrutura (append, remove, acesso, interseção, etc.).
- **Condicionais e Lógica:** Utilização de `if/else` combinado com operadores lógicos para tomada de decisão.
- **Entrada do Usuário:** Exercícios com `input()` para tornar os programas interativos.
- **Formatação de Saída:** Uso de f-strings e formatação numérica (`:.2f`) para exibição profissional.
- **Erro Controlado:** Exercício 22 demonstra propositalmente um erro para fixar o conceito de imutabilidade de tuplas.
