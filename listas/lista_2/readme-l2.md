# ✊📄✂️ Jokenpo - Versão 1 Jogador (vs CPU)

## 🌟 Sobre o Projeto  
Este é o **Exercício 2 da Lista 2** do curso **"Fundamentos de Linguagem Python - Do Básico a Aplicações de IA"** da **Data Science Academy**, porém com uma adaptação pessoal: o jogo foi desenvolvido do zero por mim e ajustado para a experiência clássica de **um jogador contra o computador**.

A proposta original foi expandida com minha própria implementação, mantendo a simplicidade do terminal, mas garantindo uma execução limpa, interativa e fiel às regras do jogo.

- 🎮 Interface direta no terminal
- 🤖 Oponente controlado pela CPU com jogadas aleatórias
- ✅ Validação básica da entrada do usuário
- 📊 Lógica completa de vitória, derrota e empate
- 🚀 Código autoral, leve e sem dependências externas

---

## 🎯 Objetivo do Exercício (com minha adaptação)  
Praticar lógica de programação com Python utilizando:

- Estruturas de decisão (`if/elif/else`)
- Geração de opções aleatórias com `random.choice()`
- Interação com o usuário via terminal
- Manipulação de strings (`lower().strip()`)
- Organização de código autoral e funcional

## 🔥 Destaques da Minha Implementação

- **Código autoral:** Desenvolvido do zero por mim, com adaptações para tornar a experiência mais fluida.
