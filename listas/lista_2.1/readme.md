# 🐍 Fundamentos de Python - Lista 2 de Exercícios (DSA)

## 🌟 Sobre o Projeto  
Lista de exercícios avançados do curso **"Fundamentos de Linguagem Python - Do Básico a Aplicações de IA"** da **Data Science Academy**. O foco é aprofundar os conceitos de funções, estruturas de dados, list comprehension, funções lambda, ordenação e tratamento de exceções, com aplicações práticas como jogos e análise de dados simples.

- 🎯 10 exercícios que exploram funções, condicionais, laços, manipulação de listas, dicionários, ordenação e programação funcional (map, lambda)
- 📓 Desenvolvido em Jupyter Notebook para experimentação interativa
- ✅ Soluções completas com exemplos de entrada/saída
- 🧠 Conceitos intermediários de Python aplicados em problemas reais

## 🚀 Funcionalidades  

| **Exercício** | **Descrição**                                                                 | **Conceitos Aplicados**                  |  
|---------------|-------------------------------------------------------------------------------|-------------------------------------------|  
| 1️⃣            | Classificação de triângulos (Equilátero, Isósceles, Escaleno)                 | Funções, condicionais, validação         |  
| 2️⃣            | Tabuada de um número com list comprehension                                   | List comprehension, laços, formatação    |

| 5️⃣            | Cálculo do IMC (Índice de Massa Corporal)                                     | Funções, operadores aritméticos          |  
| 6️⃣            | Ordenação de lista de pessoas por idade usando `sorted` e `lambda`           | Ordenação, funções lambda                |

## 🔥 Destaques Técnicos

- **Funções bem definidas:** Cada exercício é encapsulado em funções reutilizáveis, promovendo boas práticas de modularização.
- **List Comprehension:** Uso extensivo para criar listas de forma concisa (tabuada, quadrados pares, filtro de e-mails).
- **Programação funcional:** Emprego de `map` e `lambda` para transformar coleções sem loops explícitos.
- **Ordenação personalizada:** `sorted` com funções lambda para ordenar dicionários por chaves específicas.
- **Tratamento de exceções:** No jogo de adivinhação, `try/except` garante que entradas inválidas não interrompem o programa.
- **Estruturas de dados:** Dicionários são usados para armazenar contagens (pares/ímpares) e dados de pessoas.
- **Aleatoriedade:** Módulo `random` para gerar o número secreto, tornando cada partida única.
- **Iteração com `for/else`:** No jogo, o `else` do `for` é acionado quando todas as tentativas se esgotam sem sucesso, uma construção elegante e pouco explorada.
